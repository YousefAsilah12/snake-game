    <h1>Snake Game</h1>
    <p>This is a classic Snake game where the player controls a snake that moves around the game board collecting fruits. The game is built using HTML, CSS, and JavaScript.</p>
    <h2>How to Play</h2>
    <p>The objective of the game is to eat as many fruits as possible without colliding with the walls or the snake's body. The game ends when the snake collides with either the wall or its own body.</p>
    <ul>
      <li>To move the snake, use the arrow keys on your keyboard: up, down, left, and right.</li>
      <li>Each time the snake eats a fruit, its body grows longer, making it harder to navigate.</li>
    </ul>
    <h2>Features</h2>
    <ul>
      <li>Simple and intuitive gameplay</li>
      <li>Randomly generated fruits for endless gameplay</li>
      <li>Scoreboard to keep track of your highest score</li>
      <li>Mobile-friendly design</li>
    </ul>
    <h2>Future Features</h2>
    <ul>
      <li>Add multiple levels with increasing difficulty</li>
      <li>Implement power-ups or obstacles to add variety to gameplay</li>
      <li>Include a leaderboard to compare scores with other players</li>
      <li>Allow customization of snake appearance</li>
    </ul>
    <h2>Development</h2>
    <p>This game was developed using HTML, CSS, and JavaScript. The game board and fruits are generated dynamically using JavaScript. Sound effects are added using the HTML5 audio element.</p>
    <p>Feel free to clone the repository and modify the game to your liking.</p>
